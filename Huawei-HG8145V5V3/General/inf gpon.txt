
pasos para gpon
www.realfirmware.net
1.- Entrar por web a la ont v3 ip web 192.168.1.254
	1.1- Subir XML en herramientas - respaldo / restauracion.
	1.2- Esperar a que reinicie la ont
2.- ejecutar el firmware v2
	2.1 Reiniciar cuando se queden los led prendidos(41 s aprox)
	2.2 STOP al firmware
3.- Entrar por telnet 192.168.100.1
	user=root pass=admin
	3.1 su
	3.2 shell
	3.3 EquipMode.sh on

4.- Ejecutar el firmware v2
	4.1 Reiniciar cuando se queden los led prendidos (31 seg aprox)
	4.2 STOP al firmware

5-. Entrar por telnet 192.168.100.1
	user=root pass=admin
	5.1 su
	5.2 shell
	5.3 sudo restorehwmode.sh
        
	5.5 EquipMode.sh off
	5.6 Reboot

para activar el remoto despues que la ont esta liberada
subir el xml subir en azul y no tocar nada del 
remoto en la ont ya que esta configurado para que lea
cualquier wan que se le ponga y funcione el remoto.
